# 🌐 Emre Kocabey - Kişisel Web Sitesi

Bu proje, HTML, CSS ve JavaScript kullanılarak oluşturulmuş kişisel web sitemdir. Amacım, kendimi tanıtmak, projelerimi sergilemek ve iletişim bilgilerimi paylaşmaktır.

## 🚀 Özellikler

- **Ana Sayfa:** Kısa bir özgeçmiş ve hoş geldiniz mesajı.
- **Hakkımda:** Eğitim, deneyim ve ilgi alanlarım hakkında bilgiler.
- **Projeler:** Üzerinde çalıştığım veya tamamladığım projelerin listesi.
- **İletişim:** E-posta ve sosyal medya hesaplarıma yönlendiren bağlantılar.

## 🛠️ Kullanılan Teknolojiler

- **HTML5:** Sayfa yapısını oluşturmak için.
- **CSS3:** Stil ve düzenlemeler için.
- **JavaScript:** Etkileşimli öğeler ve dinamik içerikler için.
